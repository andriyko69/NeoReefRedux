# NeoReefRedux

NeoReefRedux enhances coral reef world generation in Minecraft by replacing flat vanilla reefs with more natural formations featuring rocky outcrops and varied underwater terrain.

## What This Mod Does

- Generates coral reefs on rocky structures instead of flat seabeds
- Adds more natural reef shapes and terrain variation
- Improves overall underwater visual depth while staying vanilla-friendly
- Includes a config option `disableVanillaCorals` (default: false) to disable vanilla coral reef generation

## What’s Different From the Original

NeoReefRedux is an update of Reef Redux for:

- Minecraft 1.21.1
- NeoForge

No gameplay changes were made beyond updating and maintaining compatibility, aside from adding the `disableVanillaCorals` configuration option.
